# WordPress.org assets

These files are deployed to the plugin's WordPress.org SVN `assets/` directory
(they are NOT shipped inside the plugin zip).

- `icon.svg`, `banner-772x250.svg`, `banner-1544x500.svg` — brand placeholders.
  WordPress.org also accepts PNG/JPG; convert if you prefer raster.
